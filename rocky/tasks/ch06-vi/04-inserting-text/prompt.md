Read chapter 6, section 6.4 "Inserting text" (pp. 88-89): the six keys that
switch from command mode into insert mode - i and a (before/after a
character), I and A (start/end of line), and o and O (open a line
after/before).

These keys differ only by case, so the questions are multiple choice.
Remember: after inserting, press Esc to return to command mode.
