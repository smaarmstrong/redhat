BONUS TASK — requires the `flatpak` package, the Flathub remote, and internet
access (the app is downloaded from the network, so this can take a while).

Install the GNOME Calculator Flatpak application from Flathub:

  app id: org.gnome.Calculator

For example:

  flatpak install -y flathub org.gnome.Calculator

When you are done, `flatpak list` must show `org.gnome.Calculator`.
