Set up a project directory tree under /srv/proj:

  1. Create these three directories:
       /srv/proj/src
       /srv/proj/bin
       /srv/proj/docs

  2. Copy /etc/hostname to /srv/proj/docs/hostname.txt
     (the copy's content must match /etc/hostname).

  3. Create an empty file at /srv/proj/src/main.sh
